# 📊 Web-Based GPA Calculator

A simple and fast web tool designed to calculate GPA for students. It helps to track academic performance easily.

## 🛠️ Tech Stack
* **Frontend:** HTML & CSS
* **Logic:** JavaScript

## ✨ Features
* Fast and accurate GPA calculation.
* Clean, minimalist, and simple user interface.
* Responsive layout for both mobile and desktop screens.
